EmptyDownloads
==============

A simple script to empty out the downloads folder


Turns this:

<pre>
  Downloads/
  ├── bar.1
  ├── bar.2
  ├── bar.3
  └── foo
</pre>

Into this:

<pre>
  Downloads/
  └── backup
      └── 02-18
        ├── bar.1
        ├── bar.2
        ├── bar.3
        └── foo
</pre>
